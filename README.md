# Burai Live

Official website for **BURAI LIVE**, a Tokyo-based software development startup providing modern, scalable IT solutions for businesses and individuals.

This repository contains the source code for the company homepage, built with static site technologies and optimized for deployment on GitHub Pages.

---

## 🚀 About Burai Live

**BURAI LIVE** specializes in:
- Custom software development  
- Cloud integration & modern infrastructure  
- IT consulting and solution design  
- Web application development  
- Technical product support  

Our goal is to create clean, efficient, and reliable technology that empowers clients to move faster.

---

## 📁 Project Structure

burai-live-site/
├─ _config.yml # Jekyll site configuration
├─ index.md # Homepage
├─ about/ # Company info pages
├─ services/ # Service descriptions
├─ assets/ # Images, CSS, JS
└─ _posts/ # Blog/news (optional)


---

## 🛠️ Tech Stack

- **Jekyll** – Static site generator  
- **GitHub Pages** – Hosting  
- **Markdown** – Content writing  
- **HTML/CSS/JS** – UI customization  

---

## 🔧 Local Development

Make sure you have Ruby and Jekyll installed.

```bash
bundle install
bundle exec jekyll serve
```
