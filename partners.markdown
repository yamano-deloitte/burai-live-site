# パートナー

Burai Liveは、DX・AI・ドローン・Web開発など多様な分野のパートナー企業・団体と連携し、
お客様に最適なソリューションを提供しています。

## パートナーシップについて

現在、パートナー企業・団体の情報を準備中です。  
ご興味のある企業様・団体様は[お問い合わせ](/contact/)よりご連絡ください。

## パートナー募集

Burai Liveでは、以下のような企業・団体様とのパートナーシップを歓迎しています。

- AI・機械学習の技術を持つ企業・研究機関
- クラウドインフラ・セキュリティ分野のベンダー
- ドローン・UAVのハードウェアメーカー・商社
- 医療・ヘルスケア分野のシステムインテグレーター
- 新規事業・スタートアップ支援に関心のある投資家・VC

ご提携のご相談は[お問い合わせ](/contact/)フォームよりお気軽にどうぞ。
